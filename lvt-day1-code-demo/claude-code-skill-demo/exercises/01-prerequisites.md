# Exercise 1: Prerequisites

## Goal

Get your environment ready so the rest of the walkthrough runs smoothly.

## What to do

1. Confirm Claude Code is installed and running. The `claude` command should be
   available in your terminal.
2. Create a working directory for the exercise and move into it:

   ```bash
   mkdir -p ~/skill-lab && cd ~/skill-lab
   ```

## What to observe

- The `claude` command starts an interactive session. If it does not, install or
  update Claude Code before continuing.
- `~/skill-lab` is only a scratch directory for running the review. The Skill you
  build in the next exercise lives somewhere else, in your home directory, so it is
  available across every project.

## Personal Skill

This exercise builds a personal Skill. It lives in your home directory under
`~/.claude/skills` and is available across all of your projects, not just this one.

## Learning outcome

You understand the difference between a working directory for a task and the home
directory location where a personal Skill is stored.
